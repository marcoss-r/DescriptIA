# DescriptIA

Juego de tarjetas por equipos, para jugar pasándose el móvil. Ver el plan completo en [`PLAN_DESARROLLO.md`](PLAN_DESARROLLO.md).

## Cómo abrir el proyecto

Basta con abrir `index.html` con el navegador (doble clic, o arrastrarlo a una pestaña).

Si el navegador se queja de permisos al cargar los `data/*.js`, sirve la carpeta con un servidor local, por ejemplo:

```bash
npx serve .
```

y abre la URL que te indique (normalmente `http://localhost:3000`).
